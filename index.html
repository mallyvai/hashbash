hashbash by mallyvai

Using genetic algorithms to breed hash functions.

HashBash was a project for a cryptography class I took about two years ago. The NIST competition for a new hash function had an approaching deadline, so I figured I may as well try and see if I could come up with any novel way to generate hash functions, and came to the idea of using a custom assembly language and genetic algorithms to try and breed one. Unfortunately, the algorithm couldn't effectively generate a useful hash function (just some checksums, really). The paper goes into more detail. Additionally, the code itself may or may not work properly without err'ing somewhere - I finished the paper, made a few accidental modifications to the master branch, and never bothered to fix it- but I'm leaving it up here as an example of how things may be approached, what methods I used to get the results I did, and (hopefully) provide a base for anyone trying to improve on this idea in the future. :-) The paper has detailed information concerning my rationale, approach, and outcomes. It also identifies (many) possible areas for improvement.
